## 1、backgrounds

CSS 属性定义背景效果:

- background-color

  - ``````
    body {background-color:#b0c4de;}
    ``````

- background-image

  - ```body {background-image:url('paper.gif');}```

- background-repeat

- background-attachment

- background-position
